# bot_sync — Local bot → Vercel dashboard telemetry

Drop this folder into your PolyMarket bot repo (or install from this repo).
It's **stdlib-only** (urllib) — no extra dependencies on the bot side.

## 1. Install

Copy the `bot_sync/` folder into your bot project root:

```
your_bot/
  bot_sync/
    __init__.py
    supabase_sync.py
  run_bot.py
  ...
```

## 2. Configure environment

In your bot's `.env`:

```env
DASHBOARD_URL=https://botdashboard.vercel.app
BOT_API_KEY=bot_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxx   # generated by scripts/create_user.py
```

## 3. Wire it into the bot

Add to `run_bot.py` (adapt to your actual variable names):

```python
from bot_sync import DashboardSync

class PolyMarketBot:
    def __init__(self, ...):
        ...
        self.sync = DashboardSync()   # reads DASHBOARD_URL + BOT_API_KEY from env

    def run_cycle(self, cycle: int):
        ...
        # At the END of every cycle, push status:
        self.sync.push_status({
            "running": True,
            "dry_run": self.settings.dry_run,
            "scale_level": {
                "id": self.risk_manager.current_level.id,
                "base_bet": self.risk_manager.current_level.base_bet_usd,
                "min_pnl_required": self.risk_manager.current_level.min_pnl_required,
                "next_unlock": self._next_level_info(),
            },
            "next_cycle_at": (datetime.now(timezone.utc) + timedelta(minutes=5)).isoformat(),
            "net_pnl": self.journal.get_total_pnl(),
            "cycle": cycle,
        })

    def on_trade_placed(self, trade):
        self.sync.push_trade({
            "trade_id": trade.id,
            "timestamp": trade.timestamp.isoformat(),
            "asset": trade.asset,
            "direction": trade.direction,
            "entry_price": trade.entry_price,
            "size_usd": trade.size_usd,
            "shares": trade.shares,
            "confidence": trade.confidence,
            "status": "PLACED",
            "end_time": trade.end_time.isoformat(),
        })

    def on_trade_resolved(self, trade, outcome: str, pnl: float):
        # Same trade_id → upserts the existing row
        self.sync.push_trade({
            "trade_id": trade.id,
            "timestamp": trade.timestamp.isoformat(),
            "asset": trade.asset,
            "direction": trade.direction,
            "entry_price": trade.entry_price,
            "size_usd": trade.size_usd,
            "confidence": trade.confidence,
            "status": "CLOSED",
            "outcome": outcome,              # "WIN" | "LOSS"
            "pnl": pnl,
            "resolved_at": datetime.now(timezone.utc).isoformat(),
            "end_time": trade.end_time.isoformat(),
        })
```

## 4. Safety

- All calls are **try/except** — the sync can never crash your bot.
- Errors are throttled to max 1 log per minute.
- Timeouts default to 8s with 2 retries.
- Use `DashboardSync(silent=True)` if you don't want any logs at all.

## 5. Data flow

```
[your bot]  ─push─>  /api/bot/push (Bearer: BOT_API_KEY)
                       │
                       ▼
                 Supabase Postgres
                       │
                       ▼
[dashboard]  ─read─>  /api/summary, /api/trades, ... (session cookie)
```

Each user's data is isolated by `user_id` — the `BOT_API_KEY` identifies
which tenant you are.
